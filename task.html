Block Layout

Block elements take the full width available and stack vertically they always start on a new line.

display:block

These elements have display:block applied:

Block Element 1
Block Element 2
Block Element
/* CSS for block elements */ .block-item { display: block; /* Makes element a block */ background-color: #4a90e2; color: white; padding: 15px; margin: 10px 0; /* Vertical margin works */ width: 100%; /* Takes full width by default */ }
Default Block Elements : <div>, <h1>-<h6>, <p>, <section>, <article>
